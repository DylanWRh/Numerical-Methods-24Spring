# 数值方法 Homework 8

(1) 直接用`fft`计算的代码如下

```matlab
x = [1, 3, 5, 6, 4, 2, 2, 7];

X_fft = fft(x);
disp(transpose(X_fft));
```
得到结果为
```
  30.0000 + 0.0000i
  -1.5858 - 3.0000i
  -2.0000 + 8.0000i
  -4.4142 + 3.0000i
  -6.0000 + 0.0000i
  -4.4142 - 3.0000i
  -2.0000 - 8.0000i
  -1.5858 + 3.0000i
```

(2) 用离散傅里叶变换矩阵计算的代码为
```matlab
x = [1, 3, 5, 6, 4, 2, 2, 7];

N = length(x);
n = 0:N-1;
k = 0:N-1;
F = exp(-1i*2*pi/N * n' * k);

X_matrix = F * x';
disp(X_matrix);
```

得到结果为
```
  30.0000 + 0.0000i
  -1.5858 - 3.0000i
  -2.0000 + 8.0000i
  -4.4142 + 3.0000i
  -6.0000 - 0.0000i
  -4.4142 - 3.0000i
  -2.0000 - 8.0000i
  -1.5858 + 3.0000i
```

比较两个结果的差值以及差值的范数
```matlab
disp(transpose(X_fft) - X_matrix);
disp(norm(transpose(X_fft) - X_matrix))
```
结果为
```
   1.0e-13 *

   0.0000 + 0.0000i
   0.0111 + 0.0000i
   0.0355 + 0.0000i
   0.0000 - 0.0222i
   0.0000 + 0.0514i
   0.0000 + 0.0711i
   0.0666 + 0.0000i
  -0.2109 - 0.1599i

   2.8994e-14
```

可以认为两种方法求出的结果相等
